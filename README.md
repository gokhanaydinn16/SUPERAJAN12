# Otonom Kripto Ajan Sistemi

Bu repo, deterministik risk ve yürütme çekirdeği üstüne kurulu bir otonom kripto trading işletim sistemi MVP'sidir.

Ana hedefler:

- venue-aware piyasa durumu doğrulama
- sert risk limitleri ve kill switch
- araştırma ajanlarından gelen önerileri kontrollü kullanma
- açıklanabilir karar döngüsü
- operatör paneli ve olay kaydı

Bu sistem ilk aşamada bağımlılıksız Python standart kütüphaneleri ile çalışacak şekilde tasarlanır.
